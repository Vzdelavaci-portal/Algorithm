# 🚀 Algorithm Lab

Modern interactive algorithm visualizations built with HTML, CSS and JavaScript.

Algorithm Lab is an educational project focused on:
- algorithm visualization
- interactive simulations
- modern UI/UX
- making programming easier to understand

The goal of the project is to present algorithms:
- visually
- interactively
- in a fun way
- beginner friendly

The project combines:
- algorithm animations
- modern interface design
- educational explanations
- real-time simulations

---

# ✨ Features

- 🎮 Interactive controls
- ⚡ Real-time animations
- 🌍 Multi-language support
- 🎨 Modern neon UI
- 📱 Responsive layout
- 🧠 Educational explanations
- 🎥 Social media friendly visuals

---

# 🛠 Technologies

- HTML5

- JavaScript

Pure frontend project without frameworks.

# 💡 Goal of the Project

Many people see algorithms only as:
- theory

- complex code

Algorithm Lab tries to show that algorithms can be:
- visual
