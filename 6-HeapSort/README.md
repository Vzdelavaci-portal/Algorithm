# 🇬🇧 About

Heap Sort is an efficient sorting algorithm that uses a special data structure called a **Binary Heap**.

Unlike many other sorting algorithms, Heap Sort first organizes data into a tree-like structure and then repeatedly extracts the largest value.

This visualization helps users understand:
- how a Binary Heap works
- how a Max Heap is built
- how Heapify operates
- how Heap Sort sorts data efficiently

# 🧠 How Heap Sort Works

Heap Sort consists of two main phases.

### Step 1 — Build Max Heap

The algorithm transforms the array into a Max Heap.

Example:

```txt
        90
      /    \
    70      60
   /  \    /  \
 40  30  20  10
```

The largest value is always located at the root.

---

### Step 2 — Sort

The root is swapped with the last element.

```txt
90 ↔ 10
```

The heap is then rebuilt and the process repeats until all values are sorted.

---

# 🌳 Why Use a Heap?

A heap allows efficient access to the largest element.

Instead of searching the entire array every time, the maximum value is always available at the root.

This makes Heap Sort very efficient.

# 🧠 Jak Heap Sort pracuje?

### Krok 1 — Vytvoření Max Heap

Pole se převede na binární haldu.

```txt
        90
      /    \
    70      60
   /  \    /  \
 40  30  20  10
```

Největší hodnota je vždy v kořeni.

---

### Krok 2 — Třídění

Kořen se přesune na konec pole.

```txt
90 ↔ 10
```

Poté se halda znovu upraví a celý proces se opakuje.

---

# 🌳 Proč používat haldu?

Halda umožňuje velmi rychle získat největší hodnotu.

Není potřeba procházet celé pole — maximum je vždy na vrcholu stromu.

Právě díky tomu je Heap Sort velmi efektivní.
