# 🫧 Selection Sort Visualization

Interactive visualization of the Selection Sort algorithm built with HTML, CSS and JavaScript.

This project is part of the **Algorithm Lab** series focused on creating modern and educational algorithm simulations.

---

# 🇬🇧 About

Selection Sort is a simple sorting algorithm that repeatedly:
1. finds the smallest value
2. moves it to the correct position
3. grows the sorted part of the array step by step

This visualization helps users understand:
- how Selection Sort works
- how values are compared
- how the minimum value is selected
- how swapping changes the array

# 🇨🇿 Jak funguje Selection Sort?

Selection Sort je jednoduchý třídicí algoritmus, který opakovaně:
1. najde nejmenší hodnotu
2. přesune ji na správné místo
3. postupně zvětšuje seřazenou část pole

Vizualizace pomáhá pochopit:
- jak algoritmus funguje
- jak probíhá porovnávání hodnot
- jak se hledá minimum
- jak výměna mění pole
