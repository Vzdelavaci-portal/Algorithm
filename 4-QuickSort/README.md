# ⚡ Quick Sort Visualization

Interactive visualization of the Quick Sort algorithm built with HTML, CSS and JavaScript.

This project is part of the **Algorithm Lab** series focused on creating modern and educational algorithm simulations.

---

# 🇬🇧 About

Quick Sort is one of the most popular and efficient sorting algorithms.

It works by:
1. selecting a pivot value
2. partitioning the array around the pivot
3. recursively sorting the smaller partitions

This visualization helps users understand:
- how Quick Sort works
- what a pivot is
- how partitioning works
- how recursive sorting builds a sorted array

# 🧠 How Quick Sort Works

Quick Sort chooses a pivot value and divides the array into two groups:

- values smaller than the pivot
- values greater than the pivot

The same process is then repeated recursively for each group.

This divide-and-conquer approach makes Quick Sort one of the fastest sorting algorithms in practice.

# 🇨🇿 Jak funguje Quick Sort?

Quick Sort patří mezi nejznámější a nejrychlejší třídicí algoritmy.

Pracuje tak, že:
1. vybere pivot
2. rozdělí pole podle pivotu
3. rekurzivně seřadí obě vzniklé části

Vizualizace pomáhá pochopit:
- jak funguje pivot
- jak probíhá rozdělení pole
- jak funguje rekurze
- proč je Quick Sort tak efektivní
